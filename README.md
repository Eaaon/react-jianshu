
## 技术栈：
  react + redux + axios + antd + react-router + ES6 + Sass + immutable


## 运行项目（nodejs 6.0+）

```
 git clone https://github.com/Eaaon/react-jianshu.git

 cd jianshu

 npm i 
  
 npm start

 npm run build （发布）
```
